# WhichWayNigeria - ADC 2027 Election Command Center

AI-powered political polling analysis platform for the African Democratic Congress (ADC) 2027 presidential election campaign.

## Features

- **AI-Powered Polling Analysis**: Automated daily analysis using OpenRouter AI (Google Gemini Flash)
- **Real-time Dashboard**: Interactive charts showing polling trends
- **LGA Registration Calculator**: Calculate local government area registration performance
- **Media Coverage Aggregator**: Real-time news sentiment analysis
- **State-by-State Analysis**: Breakdown of ADC presence across Nigeria's 36 states + FCT
- **Official ADC Integration**: Direct link to official ADC registration portal
